Foundations & how computers think

Binary, logic gates, and the leap from electricity to abstraction. We build up just enough hardware intuition to make the rest of the course click.

binary logic abstraction

Programming fundamentals

Variables, control flow, functions, and the habits of writing code that other people (and future-you) can read.

syntax control-flow style
03

Data structures

Arrays, lists, maps, trees. When to reach for which, and what each one quietly costs you.

arrays trees complexity
04

Algorithms & problem solving

Searching, sorting, recursion, and a working framework for breaking unfamiliar problems apart.

recursion sorting big-O
05

Building real things

A capstone project: pick a problem you actually care about and ship something that solves it end-to-end.

project git deploy
